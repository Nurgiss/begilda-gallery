## nginx конфигурация

```nginx
server {
    listen 443 ssl http2;
    server_name begildagallery.com www.begildagallery.com;

    root /var/www/begilda-gallery/dist;
    index index.html;

    # SPA роутинг
    location / {
        try_files $uri $uri/ /index.html;
    }

    # Загруженные файлы
    location /uploads {
        alias /var/www/begilda-gallery/backend/uploads;
    }

    # API → Node.js
    location /api {
        proxy_pass http://localhost:3001;
    }
}
```

> ⚠️ `index.html` не имеет `Cache-Control: no-cache` — браузер кэширует его.  
> Решение: всегда делать **Cmd+Shift+R** после деплоя.
