# Обновление: Статус наличия для магазина и скрытые картины

## Изменения в базе данных

### 1. Модель ShopItem
Добавлено поле `availability` (Boolean):
- **true** - В наличии (по умолчанию)
- **false** - Продано

### 2. Модель Painting
Добавлено поле `hidden` (Boolean):
- **true** - Скрыто от пользователей (не удаляется, просто не отображается)
- **false** - Видимо (по умолчанию)

## Изменения в Admin панели

### ShopManager
- Добавлена секция "Статус" в форме редактирования
- Выбор между "В наличии" ✓ и "Продано" ✗
- В таблице товаров добавлена колонка "Статус" с визуальным индикатором

### PaintingsManager  
- Добавлен чекбокс "🚫 Скрыть (не удаляя, просто скрыть от пользователей)"
- В таблице картин добавлена колонка "Скрыто"
- Скрытые картины помечаются как "🚫 Да"

## Изменения на фронтенде

### Каталог картин (Catalog)
- Фильтрует скрытые картины (`hidden: true`)
- Фильтрует картины только для выставок (`exhibitionOnly: true`)

### Магазин (Shop)
- Показывает только доступные товары (`availability: true`)
- Проданные товары не отображаются в списке

### Детальные страницы

#### PaintingDetail
- Уже была поддержка статуса availability
- Кнопка "Add to Cart" отключена для проданных картин

#### ShopDetail
- Добавлено предупреждение для проданных товаров
- Кнопка "Add to Cart" заменена на "Sold Out" для проданных товаров

### Featured Paintings
- Фильтрует скрытые картины из избранных

## Использование

### Для магазина
1. Войдите в админку → Shop
2. Редактируйте товар
3. В секции "Статус" выберите:
   - "✓ В наличии" - товар виден и доступен для покупки
   - "✗ Продано" - товар скрыт из магазина

### Для картин
1. Войдите в админку → Paintings
2. Редактируйте картину
3. Установите чекбокс "🚫 Скрыть":
   - Включено - картина скрыта от пользователей (но не удалена из БД)
   - Выключено - картина видна в каталоге

## Преимущества

✅ **Для магазина**: Возможность отметить товары как проданные без удаления
✅ **Для картин**: Временно скрыть картину без потери данных
✅ **История**: Все данные сохраняются в базе, можно вернуть обратно
✅ **Гибкость**: Легко управлять видимостью без удаления записей

## Технические детали

- База данных: SQLite (Prisma ORM)
- Новые индексы для оптимизации:
  - `shop_items.availability`
  - `paintings.hidden`
- Все изменения обратно совместимы (по умолчанию все товары доступны, все картины видимы)
